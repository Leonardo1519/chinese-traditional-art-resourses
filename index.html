项目简介

非遗灵感充电站是由汕头职业技术学院艺术设计学院发起的公益性非遗资源导航平台，通过现代 Web 技术和 AI 辅助编程，为设计教育、文创研发和文化传承提供一站式资源检索服务。

🎯 发起初衷

中国非遗资源分散在各级政府、博物馆、研究机构等平台，信息碎片化、检索困难。我们希望整合资源、服务教学、激发创意、传承文化，探索"师生+AI"协同创作的新型教学模式。

📦 项目成果

网站包含 11 大板块、290+ 精选资源入口，涵盖国家级平台、地方资源库、学术研究、博物馆、名人纪念馆、国际非遗、红色专区、影视、书籍、文创项目、设计工作室，可作为高校设计专业课程的常备资源库。

🛠️ 技术架构

采用 HTML5 / CSS3 / JavaScript (ES6+) 原生技术栈构建，借助 Cursor AI 辅助编程、GitHub 代码托管与版本控制、Vercel 云端自动化部署，实现零依赖、轻量化、响应式的高性能网站。

🤖 AI 赋能传承

AI 辅助编程降低技术门槛，让设计专业学生无需深厚编程背景也能构建专业级 Web 项目。AI 还赋能数据整理、内容生成与体验优化，加速非遗文化的数字化传承与创新应用。
