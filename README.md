## 📖 项目概述

**非遗灵感充电站**是一个由汕头职业技术学院艺术设计学院发起的公益性非物质文化遗产资源导航平台，通过现代化的 Web 技术和 AI 辅助编程，系统性整合全国范围内的非遗资源，为设计教育、文创研发和文化传承提供一站式的资源检索服务。

---

## 💡 项目发起初衷

### 背景与痛点

在数字化时代，中国拥有丰富而珍贵的非物质文化遗产资源，然而这些资源分散在各级政府部门、博物馆、研究机构、高校实验室等不同平台，呈现出**信息碎片化、检索困难、获取成本高**的现状。

对于设计类专业的师生而言，在进行非遗文创设计、毕业创作或学术研究时，常常面临以下困境：

- 🔍 **资源分散**：需要花费大量时间在多个网站间跳转查找
- 📚 **信息过载**：海量信息缺乏筛选，难以快速定位权威可靠的资源
- 🎯 **学习门槛**：缺乏系统性的资源导航，初学者不知从何入手
- ⏰ **时间成本**：重复性的资源搜集工作耗费教学与创作的宝贵时间

### 发起初心

基于上述现实问题，我们发起"非遗灵感充电站"项目，希望实现以下目标：

1. **🗂️ 整合资源**  
   系统性梳理国家级、省级、市级的非遗平台及博物馆、学术机构资源，构建分类清晰的一站式导航体系

2. **🎓 服务教学**  
   为高校设计类专业的《非遗文创设计》《传统工艺与现代设计》等课程提供便捷的教学资源库，降低师生的资源获取成本

3. **🎨 激发创意**  
   为设计师与学生提供丰富的灵感来源，促进非遗元素在当代设计中的创新应用，推动传统文化的现代化转化

4. **🌏 传承文化**  
   通过数字化手段，让更多人了解、关注并参与到非遗文化的保护与传承中，增强文化自信与民族认同

5. **🤖 探索创新**  
   实践 AI 辅助编程在教育项目中的应用，探索"师生+AI"协同创作的新型教学模式

### 网站功能特性

- ✅ **智能分类导航**：按资源类型、地域、主题等多维度分类，快速定位目标资源
- ✅ **卡片式展示**：每个资源配有 Logo、名称、简介、标签与跳转链接，信息一目了然
- ✅ **筛选与统计**：博物馆、影视资源支持按地区/类型筛选，并实时显示数量统计
- ✅ **响应式设计**：完美适配桌面、平板、手机等多种设备，随时随地访问
- ✅ **交互友好**：悬浮特效、平滑过渡、描述悬浮提示，提升用户体验
- ✅ **快速访问**："返回顶部"按钮、友情链接、二维码等便捷功能

本平台可作为设计类院校师生进行非遗相关课程的**常备工具**，适用于：

- 📚 **课程教学**：为《非遗文创设计》《传统工艺与现代设计》《品牌设计》等课程提供案例库与素材库

## 🛠️ 技术架构

### 前端技术栈

本项目采用**轻量化、原生化**的现代前端技术栈，无需复杂框架，保证网站的高性能与易维护性：

| 技术 | 版本/规范 | 应用说明 |
|------|----------|----------|
| **HTML5** | HTML Living Standard | 使用语义化标签（`<header>`, `<main>`, `<section>`, `<footer>`），提升可访问性与SEO友好度 |
| **CSS3** | CSS3 完整规范 | 采用 CSS 变量、Flexbox、Grid 布局，实现响应式设计与暗色模式兼容 |
| **JavaScript** | ES6+ (ECMAScript 2015+) | 模块化开发（ES Modules），动态渲染页面内容，实现筛选、统计等交互功能 |
| **Google Fonts** | Noto Sans SC / Space Grotesk | 优化中英文混排显示效果，提升阅读体验 |

### 开发工具与流程

- **🖥️ Cursor AI IDE**  
  使用 Cursor 智能编程工具，借助 AI 辅助代码生成、调试与优化，显著提升开发效率。通过自然语言对话即可生成页面结构、样式代码与交互逻辑，实现"师生+AI"的协同创作模式。

- **📦 GitHub 代码托管与版本控制**  
  项目代码托管于 GitHub 平台（`chinese-traditional-art-resourses`），使用 Git 进行版本控制，支持多人协作、代码回滚与历史追溯。所有代码开源透明，便于社区参与与持续改进。

- **☁️ Vercel 云端部署**  
  采用 Vercel 平台进行自动化部署，每次代码推送至 GitHub 主分支后，Vercel 自动触发构建与发布流程，实现**零配置、秒级部署**。全球 CDN 加速，确保国内外用户的快速访问。

- **🎨 设计规范**  
  - 采用组件化设计思路，卡片式布局便于信息展示与扩展
  - 以米黄、橙色为主色调，传递传统文化的温润质感
  - 响应式断点：桌面端 (1200px+)、平板 (768-1200px)、手机 (< 768px)
  - 遵循无障碍设计原则（WCAG），支持键盘导航与屏幕阅读器

### AI 在项目开发中的应用

本项目是**AI 辅助编程在教育领域的创新实践**，展示了人工智能如何助力非遗文化的数字化传承：

#### 1. **AI 辅助编程，降低技术门槛**

传统的 Web 开发需要设计师具备深厚的编程基础，而通过 **Cursor AI IDE** 的智能代码生成功能，师生可以：

- 💬 **自然语言编程**：用中文描述需求，AI 自动生成 HTML/CSS/JavaScript 代码
- 🐛 **智能调试**：AI 自动识别代码错误并提供修复建议
- 📝 **代码补全**：基于上下文智能预测下一步代码，减少重复劳动
- 🎨 **样式优化**：AI 根据设计稿自动生成响应式 CSS 代码

**案例**：在本项目中，学生只需描述"创建一个卡片网格布局，每个卡片包含 Logo、标题、描述和跳转按钮"，AI 即可生成完整的 HTML 结构与 CSS 样式，大幅降低技术门槛。

#### 2. **数据整理与结构化**

非遗资源的收集与整理是项目最耗时的环节。通过 AI 技术，我们实现了：

- 📊 **自动化数据提取**：AI 辅助从官网批量提取机构名称、简介、链接等信息
- 🗂️ **智能分类打标签**：AI 根据内容自动为资源打上"国家级""博物馆""非遗传承人"等标签

**成果**：原本需要数周完成的 290+ 资源数据整理工作，在 AI 辅助下缩短至数天。

#### 3. **内容生成与优化**

- ✍️ **描述文案优化**：AI 根据官方简介生成更简洁、信息密度更高的卡片描述
- 🏷️ **标签提取**：AI 自动从描述中提取关键词作为标签（如"手工艺""非遗保护""文创设计"）

#### 4. **用户体验设计**

- 🎨 **配色方案生成**：AI 根据"传统文化""温暖""亲和"等关键词推荐色彩搭配
